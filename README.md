<h1 align="center">MATFIT</h1>

<h3><strong>Este é um aplicativo pessoal destinado aos proprietários de academias. O aplicativo tem como objetivo facilitar a administração de clientes, funcionários e equipamentos.</strong></h3>

<h4><strong>Esta aplicação consome uma API que foi desenvolvida com Java e Spring Boot.</strong></h4>

<h2>Funcionalidades:</h2>
<ul>
    <li>Realizar login</li>
    <li>Realizar logout</li>
    <li>Cadastrar alunos</li>
    <li>Visualizar alunos</li>
    <li>Excluir alunos</li>
    <li>Atualizar dados de alunos</li>
    <li>Confirmar pagamento de alunos</li>
    <li>Cadastrar funcionários</li>
    <li>Visualizar funcionários</li>
    <li>Excluir funcionários</li>
    <li>Atualizar dados de funcionários</li>
    <li>Cadastrar equipamentos</li>
    <li>Visualizar equipamentos</li>
    <li>Excluir equipamentos</li>
    <li>Atualizar dados de equipamentos</li>
</ul>
